debugout.js  

debugout.js (debug output) generates a text document from console.logs that can be emailed, downloaded, POSTed, and more. See examples and usage on GitHub.

Download on GitHub

Create a new debugout object and replace all your console log methods with debugout's log() method:

var bugout = new debugout();
bugout.log('A date:');
bugout.log(d);
bugout.log('An array:');
bugout.log(arr);
bugout.log('A function:');
bugout.log(bugout.objectSize);
bugout.log('A regex');
bugout.log(rgx);
bugout.log('an object with nested objects...');
bugout.log(obj);
bugout.log('an array of objects...');
bugout.log(arrayWithObjects);

Your application can access the log any time by using debugout's getLog() method, then you can do something with it, like letting the user download it.
